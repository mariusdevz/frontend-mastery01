# Cadence Landing Page

Cadence is a modern and responsive product landing page built with HTML and CSS.

This project showcases key front-end development concepts including:

- Responsive design with media queries
- CSS custom properties (variables)
- Dark/Light theme switching
- Flexbox layouts
- CSS animations and transitions
- Mobile-first optimization
- Clean and reusable code structure

The landing page promotes a productivity-focused application called Cadence, featuring a hero section, feature cards, call-to-action buttons, and an email signup area.

### Technologies Used

- HTML5
- CSS3
- Tabler Icons

### What I Learned

- Building responsive layouts for desktop, tablet, and mobile
- Organizing CSS with variables and reusable design tokens
- Creating theme toggles and interactive UI effects
- Structuring a professional landing page from scratch

This project represents another step forward in my front-end development journey.
